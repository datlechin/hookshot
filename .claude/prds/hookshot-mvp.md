# Hookshot MVP

A webhook testing and debugging tool built with Rust (Axum) backend and React frontend.

## Features

- Webhook capture with full request storage
- REST API for endpoint and request management
- WebSocket server for real-time notifications
- React frontend with shadcn/ui
- Single binary deployment with embedded frontend
- Cross-platform support (macOS, Linux, Windows)

## Tasks

1. Backend foundation: Database, models, and basic server setup
2. Webhook capture handler with full request storage
3. REST API for endpoint and request management
4. WebSocket server for real-time request notifications
5. Frontend scaffolding with React, TypeScript, Vite, and shadcn/ui
6. Dashboard and Endpoint View pages with real-time request list
7. Request detail view and filtering/search functionality
8. Custom response configuration UI with templates
9. Frontend embedding and single binary build
10. Documentation, CI/CD, and cross-platform release builds
