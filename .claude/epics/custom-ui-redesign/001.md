# Task: Setup Design System Foundation

## Description

Create the CSS architecture foundation for the custom design system including CSS variables for theming, typography setup with Geist Mono font, global styles, and terminal-inspired color palette.

This is the foundational task that establishes the design tokens and styling infrastructure that all other components will use.

## Acceptance Criteria

- [x] CSS variables file created with complete color palette (background, text, accent, status colors)
- [x] Typography scale defined (text-xs through text-xl with line heights)
- [x] Spacing scale established (xs: 2px through 2xl: 24px)
- [x] Geist Mono font loaded from CDN with JetBrains Mono fallback
- [x] Global reset/normalize styles applied
- [x] Dark theme fully defined (GitHub Dark Dimmed aesthetic)
- [x] CSS Modules configuration verified in Vite

## Technical Details

**`frontend/src/styles/global.css`** - Global styles and reset
```css
@import './tokens.css';

/* Font loading */
@import url('https://fonts.googleapis.com/css2?family=Geist+Mono:wght@400;500;600&display=swap');

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: var(--font-mono);
  font-size: var(--text-base);
  line-height: var(--line-height-normal);
  background: var(--bg-primary);
  color: var(--text-primary);
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

/* Focus visible styles */
*:focus-visible {
  outline: 2px solid var(--border-focus);
  outline-offset: 2px;
}

/* Selection styles */
::selection {
  background: var(--accent);
  color: white;
}
```

**Import in `frontend/src/main.tsx`**:
```tsx
import './styles/global.css'
```

### Implementation Steps

1. Create `frontend/src/styles/` directory
2. Create `tokens.css` with all design variables
3. Create `global.css` with reset and base styles
4. Load Geist Mono font from Google Fonts (or Vercel CDN)
5. Import `global.css` in `main.tsx`
6. Verify CSS Modules work with Vite (should work out of the box)
7. Test dark theme colors in browser DevTools

### Verification

- Open app in browser, inspect computed styles
- Verify `--bg-primary` is `#0d1117`
- Verify Geist Mono font loads (check Network tab)
- Check fallback works if font fails to load
