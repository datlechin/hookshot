## Description

Replace shadcn/ui Tabs component with custom Tabs component from Task 003. The RequestDetail modal uses Tabs to switch between Headers, Body, and Raw views.

## Technical Details

### Files to Modify

**`frontend/src/components/RequestDetail.tsx`**

Before (shadcn Tabs):
```tsx
import { Tabs, TabsContent, TabsList, TabsTrigger } from '@/components/ui/tabs'

// Inside Modal content:
<Tabs defaultValue="headers" className="w-full">
  <TabsList className="grid w-full grid-cols-3">
    <TabsTrigger value="headers">
      Headers ({Object.keys(request.headers).length})
    </TabsTrigger>
    <TabsTrigger value="body">Body</TabsTrigger>
    <TabsTrigger value="raw">Raw</TabsTrigger>
  </TabsList>

  <TabsContent value="headers" className="space-y-2">
    {/* Headers content */}
  </TabsContent>

  <TabsContent value="body" className="space-y-2">
    {/* Body content */}
  </TabsContent>

  <TabsContent value="raw" className="space-y-2">
    {/* Raw content */}
  </TabsContent>
</Tabs>
```

After (custom Tabs):
```tsx
import { Tabs } from '@/components/ui'

// Prepare tabs data
const tabs = [
  {
    id: 'headers',
    label: `Headers (${Object.keys(request.headers).length})`,
    content: (
      <div className="space-y-2">
        {/* Headers content */}
      </div>
    ),
  },
  {
    id: 'body',
    label: 'Body',
    content: (
      <div className="space-y-2">
        {/* Body content */}
      </div>
    ),
  },
  {
    id: 'raw',
    label: 'Raw',
    content: (
      <div className="space-y-2">
        {/* Raw content */}
      </div>
    ),
  },
]

// Use Tabs component
<Tabs tabs={tabs} defaultTab="headers" />
```
